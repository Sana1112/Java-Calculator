# SimpleCalculatorJava

A simple desktop calculator built in Java using Swing. Supports basic arithmetic, decimals, chained operations, reciprocal and error handling.

## How to run
1. Download `Calculator.java`.
2. Compile: `javac Calculator.java`
3. Run: `java Calculator`

## Features
- Modern UI
- Supports chained operations (e.g., 1+2+3*4)
- Shows errors for division by zero or invalid input
